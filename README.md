# Trabalho - Análise e Projeto de Sistemas

## Questão 07 – Lista de Compras
Foi desenvolvido um sistema com as classes ListaCompras e Produto.
O relacionamento entre elas é de 1 para N, onde uma lista pode conter vários produtos.

## Questão 08 – Coleção de CDs
Foram modeladas as classes CD e Artista.
Um artista pode possuir vários CDs.

## Questão 09 – Coleção de CDs (Variação A)
O sistema foi expandido para suportar múltiplos artistas e músicas.
Existe um relacionamento N:N entre CD e Artista.

## Questão 10 – Sala de Reunião
Foram modeladas as classes Sala, Funcionário, Reunião e Agenda.
O sistema permite agendamento e consulta de disponibilidade.

## Questão 11 – Herança
Foi criada a superclasse Pessoa, com herança para Funcionário e Cliente.
