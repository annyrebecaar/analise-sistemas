Questão 08 – Coleção de CDs

Classes:
- CD
- Artista

Atributos:

CD:
- titulo
- anoLancamento

Artista:
- nome

Métodos:

CD:
- cadastrarCD()

Artista:
- cadastrarArtista()

Relacionamento:
- Artista (1) -------- (N) CD

Requisitos Funcionais (RF):
- RF01: O sistema deve permitir cadastrar CDs.
- RF02: O sistema deve permitir cadastrar artistas.
- RF03: O sistema deve listar todos os CDs cadastrados.
- RF04: O sistema deve permitir consultar CDs por artista.

Requisitos Não Funcionais (RNF):
- RNF01: O sistema deve permitir consultas rápidas.
- RNF02: O sistema deve evitar duplicidade de registros.
- RNF03: O sistema deve ter interface simples.
